
<Steps>

# More about Selector

| Name                  |    Example    |
| :-------------------- | :-----------: |
| Element selector      |      `p`      |
| Class selector        | `.class-name` |
| ID selector           |  `#id-name`   |
| Attribute selector    |  `img[src]`   |
| Pseudo-class selector |   `p:hover`   |
