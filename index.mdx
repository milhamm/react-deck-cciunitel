<TitleWithDescription title="List and Keys">

Let‚Äôs review how you transform lists in JavaScript.

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((number) => number * 2);
console.log(doubled);
// [2, 4, 6, 8, 10]
```

`map()` method allows you to iterate over an array and modify its elements using a callback function then return the new array

</TitleWithDescription>

---

<TitleWithDescription title="List and Keys (Cont.)">

We can do the same for JSX elements

```js
const numbers = [1, 2, 3, 4, 5];
const listItems = numbers.map((number) => <li>{number}</li>);

return <ul>{listItems}</ul>;
```

</TitleWithDescription>

---

<TitleWithDescription title="List and Keys (Cont.)">

How about **keys?** Keys help react to identify any changes that happens in an item.

```js
const numbers = [1, 2, 3, 4, 5];
const listItems = numbers.map((number, index) => <li key={index}>{number}</li>);

return <ul>{listItems}</ul>;
```

Note: Using index as the keys is not recommended. For the sake of simplicity, we will use index as the keys.

</TitleWithDescription>
